# Памятка по переменным окружения (деплой)

Все настройки задаются через переменные окружения (Railway → Variables, или
`.env` на своём сервере). Ниже — что задавать, что обязательно, и что
перестаёт работать без переменной.

Легенда: 🔴 обязательно в проде · 🟡 нужно для функции · ⚪️ опционально/есть дефолт

---

## 1. Ядро платформы

| Переменная | | Назначение | Дефолт / формат |
|---|---|---|---|
| `NODE_ENV` | 🔴 | `production` включает прод-режим: строгий JWT, secure-cookie для OAuth, без dev-подсказок в ответах | `production` |
| `JWT_SECRET` | 🔴 | Секрет подписи токенов входа. **Без него сервер в проде не стартует** (иначе токены можно подделать) | длинная случайная строка (32+ симв.) |
| `DATABASE_PATH` | 🔴 | Путь к файлу базы SQLite. **Должен указывать на постоянный диск** (Railway Volume), иначе данные пропадут при рестарте | `server/utir.db` |
| `PORT` | ⚪️ | Порт сервера | `4010` |
| `APP_URL` | 🔴 | URL фронтенда — куда возвращать после OAuth-входа. В проде **обязательно задать**, иначе редирект уйдёт на localhost | `http://localhost:5173` |
| `PUBLIC_APP_URL` | 🟡 | URL фронтенда для ссылок в Telegram (открыть заказ/трек/склад) | `https://utir-soft.com` |
| `CORS_ORIGINS` | 🟡 | Разрешённые origin фронтенда через запятую. Не задан в проде → CORS открыт (с предупреждением в логе) | `https://app.tvoy-domen.kz` |

> Совет: `APP_URL` и `PUBLIC_APP_URL` обычно одинаковые — адрес твоего фронтенда.

## 2. AI (ассистент, дизайн, голосовые)

| Переменная | | Назначение | Без неё |
|---|---|---|---|
| `ANTHROPIC_API_KEY` | 🔴 | Основной ИИ (Claude Opus 4.8): ассистент, Telegram-бот, клиентский ИИ | ИИ-ассистент не отвечает |
| `ANTHROPIC_MODEL` | ⚪️ | Переопределить модель | `claude-opus-4-8` |
| `OPENAI_API_KEY` | 🟡 | AI-Дизайн (картинки) + расшифровка голосовых в боте (Whisper) | нет генерации картинок/голоса |
| `GEMINI_API_KEY` | ⚪️ | Резервный провайдер для картинок/чата | — |
| `DEEPSEEK_API_KEY` | ⚪️ | Резервный провайдер для чата | — |

---

## 3. Telegram-бот

| Переменная | | Назначение | Дефолт |
|---|---|---|---|
| `TELEGRAM_BOT_TOKEN` | 🟡 | Токен бота от @BotFather. Без него бот полностью выключен | — |
| `TELEGRAM_BOT_USERNAME` | ⚪️ | Имя бота для ссылок-приглашений | `utirsoftbot` |
| `TELEGRAM_WEBHOOK_SECRET` | 🔴* | **Безопасность.** Включает проверку источника вебхука. Без него любой, зная chat_id сотрудника, может подделать команду боту. При установке сервер сам перерегистрирует вебхук с этим секретом | случайная строка `A-Za-z0-9_-` |
| `TELEGRAM_WEBHOOK_URL` | ⚪️ | URL вебхука (если API не на `PUBLIC_APP_URL`) | `${PUBLIC_APP_URL}/api/telegram/webhook` |

> 🔴* — обязательно, **если бот подключён**. Пока `TELEGRAM_WEBHOOK_SECRET` не задан, бот работает, но вебхук уязвим к подделке.

---

## 4. Meta — WhatsApp / Instagram (после верификации бизнеса)

| Переменная | | Назначение | Дефолт |
|---|---|---|---|
| `META_VERIFY_TOKEN` | 🟡 | Verify-токен при подключении вебхука в Meta App Dashboard | `utir-verify` |
| `META_APP_SECRET` | 🔴* | **Безопасность.** Проверка подписи входящих (`X-Hub-Signature-256`). Без него можно подделать входящие сообщения в чужие чаты | app secret из Meta App |
| `META_GRAPH_VERSION` | ⚪️ | Версия Graph API | `v21.0` |

> 🔴* — обязательно **перед запуском** WhatsApp/Instagram. Токены и phone-id конкретной страницы хранятся в БД per-team (через Настройки → Интеграции), не в env.

## 5. Вход через соцсети (OAuth) — опционально

| Переменная | | Назначение |
|---|---|---|
| `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` | ⚪️ | Кнопка «Войти через Google» |
| `FACEBOOK_APP_ID` / `FACEBOOK_APP_SECRET` | ⚪️ | Кнопка «Войти через Facebook» |
| `OAUTH_CALLBACK_BASE` | ⚪️ | База для callback-URL, если API на отдельном домене (иначе берётся из запроса) |

> Не заданы → кнопки соцсетей показывают «нужна настройка». Обычный вход по email/паролю работает всегда.

---

## 6. Email и SMS

| Переменная | | Назначение | Без неё |
|---|---|---|---|
| `RESEND_API_KEY` | 🟡 | Отправка писем (приглашения в команду, сброс пароля) через Resend | письма не уходят (в dev код печатается в лог) |
| `EMAIL_FROM` | ⚪️ | Адрес отправителя | `Utir Soft <onboarding@resend.dev>` |
| `SMS_API_KEY` | ⚪️ | SMS-коды для входа по телефону | вход по телефону недоступен |

## Минимальный набор для запуска в проде

```bash
NODE_ENV=production
JWT_SECRET=<длинная случайная строка>
DATABASE_PATH=/data/utir.db          # на постоянном диске!
APP_URL=https://app.tvoy-domen.kz
PUBLIC_APP_URL=https://app.tvoy-domen.kz
CORS_ORIGINS=https://app.tvoy-domen.kz
ANTHROPIC_API_KEY=<ключ Anthropic>   # чтобы работал ИИ
```

## Что включить сразу после запуска (безопасность)

```bash
TELEGRAM_WEBHOOK_SECRET=<случайная строка>   # если подключён бот — закрывает подделку вебхука
# META_APP_SECRET=<app secret>               # перед запуском WhatsApp/Instagram
```

## Полезное для полной функциональности

```bash
OPENAI_API_KEY=<ключ>        # AI-Дизайн (картинки) + голосовые в боте
RESEND_API_KEY=<ключ>        # письма (приглашения, сброс пароля)
TELEGRAM_BOT_TOKEN=<токен>   # Telegram-бот
```
